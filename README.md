# ATS Dashboard UI

A responsive ATS (Applicant Tracking System) Dashboard built using React.js, Tailwind CSS, Axios, React Router DOM, and MockAPI.

---

# Features

## Authentication
- Login Page UI

## Dashboard
- Summary Cards
- Responsive Layout

## Jobs Management
- View Jobs
- Add Job
- Update Job
- Delete Job
- Search Jobs

## Candidates Management
- View Candidates
- Add Candidate
- Update Candidate
- Delete Candidate
- Search Candidates

## Candidate Details
- Candidate Profile
- Skills
- Score
- Experience
- Status

---

# Tech Stack

- React.js
- Tailwind CSS
- React Router DOM
- Axios
- MockAPI
- Vite


# Navigate to Project Folder

- cd React Task

# Install Dependencies

- npm install

# Start Development Server

- npm run dev

# Required Packages

- npm install axios react-router-dom formik lucide-react yup recharts tailwindcss framer-motion @tailwindcss/vite
